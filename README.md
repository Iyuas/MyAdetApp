📱 MyAdetApp – Habit Tracking iOS App
📌 Overview

MyAdetApp is an iOS application designed to help users build and maintain consistent daily habits. The app allows users to create, track, and visualize their habits over time using an intuitive and clean interface.

The goal of this project is to combine practical iOS development skills with a real-world use case focused on productivity and self-improvement.

✨ Features
✅ Create, edit, and delete habits
📅 Track daily habit completion using a calendar view
📊 Visualize progress with custom progress rings
💾 Persistent data storage using CoreData
🧭 Multi-screen navigation with UIKit
🎯 Simple and user-friendly interface
🛠️ Tech Stack
Language: Swift
Frameworks: UIKit
Persistence: CoreData
Architecture: MVC (Model-View-Controller)
🧱 Project Structure
MyAdetApp/
│
├── Controllers/        # ViewControllers (UI logic)
├── Models/             # Data models (Habit / Adet)
├── Views/              # Custom UI components
├── Managers/           # CoreData and data handling
├── Resources/          # Assets, storyboards

🚀 How It Works
Users create a habit (e.g., “Drink water”, “Study 2 hours”)
Each day, they mark the habit as completed
The app stores progress locally using CoreData
Progress is visualized:
Calendar view → shows history
Progress ring → shows consistency
