📱 MyAdetApp – Habit Tracking iOS App
📌 Overview

MyAdetApp is an iOS application designed to help users build and maintain consistent daily habits. The app allows users to create, track, and visualize their habits over time using an intuitive and clean interface.

This project focuses on applying real-world iOS development concepts such as data persistence, UI design, and app architecture.

✨ Features
Create, edit, and delete habits
Track daily habit completion using a calendar view
Visualize progress with custom progress rings
Store data locally using CoreData
Multi-screen navigation using UIKit
Simple and user-friendly interface
🛠️ Tech Stack
Language: Swift
Frameworks: UIKit
Persistence: CoreData
Architecture: MVC (Model-View-Controller)
🧱 Project Structure

MyAdetApp/

Controllers/ → ViewControllers (UI logic)
Models/ → Data models (Habit / Adet)
Views/ → Custom UI components
Managers/ → CoreData handling
Resources/ → Assets, Storyboards
🚀 How It Works
User creates a habit (e.g., “Study”, “Workout”)
Marks it as completed each day
Data is saved using CoreData
Progress is displayed through:
Calendar view (history)
Progress ring (consistency)
